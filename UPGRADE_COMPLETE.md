## 🚀 COMO COMEÇAR AGORA

### 1. Instalar
```bash
cd mobile
npm install
```

### 2. Configurar
```bash
echo "EXPO_PUBLIC_API_URL=http://192.168.86.9:3000/api" > .env.local
```

### 3. Rodar
```bash
npx expo start
# Pressione 'a' (Android) ou 'i' (iOS)
```

### 4. Testar
- ✅ Vê tela de welcome
- ✅ Clica "Login" → tela de login
- ✅ Ao logar (simulado) → home screen com drawer
- ✅ Desliza da esquerda → drawer abre
- ✅ Clica item → navega para aquela tela

**Tudo funciona!** 🎉
