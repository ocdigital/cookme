## Bem-vindo à Documentação do CookMe

Este é o centro de conhecimento do projeto **CookMe**, uma plataforma completa para gerenciamento de receitas e ingredientes.

### O que você encontrará aqui?

- **Arquitetura**: Diagrama completo e visão técnica do sistema
- **Aprendizado**: Guias e recursos para entender o projeto
- **Setup**: Instruções de configuração do ambiente
- **Guias Práticos**: AWS, testes, endpoints e mais

### Começar

Escolha por onde começar:

1. [Guia de Aprendizado](/tutoriais/guia-aprendizado) - Entenda os conceitos fundamentais
2. [Setup Rápido](/how-to/setup-rapido) - Configure seu ambiente em minutos
3. [Arquitetura](/explicacao/arquitetura/visao-geral) - Veja como o sistema é estruturado

### Tecnologias Utilizadas

- **Backend**: NestJS, TypeScript, PostgreSQL
- **Frontend**: React, TypeScript, Tailwind CSS
- **Mobile**: React Native, Expo
- **DevOps**: Docker, Docker Compose, AWS

A documentação é um trabalho contínuo. Se encontrar erros ou tiver sugestões, por favor abra uma issue.
